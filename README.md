# 🎉 Cultura na Faixa

🔗 Acesse o projeto: [https://cultura-mobile.vercel.app/](https://cultura-mobile.vercel.app/)

Uma plataforma desenvolvida em **Ionic + Angular** para reunir e divulgar **eventos gratuitos**, 
facilitando o acesso da comunidade a cultura, tecnologia, educação, networking e lazer.

---

# 📱 Sobre o Projeto

O **Cultura na faixa** é um aplicativo/mobile web criado para centralizar eventos gratuitos em um único lugar.

A proposta é permitir que usuários encontrem eventos próximos, visualizem detalhes, façam buscas por categorias e acompanhem novidades.

# ✨ Funcionalidades

- 🔍 Busca de eventos
- 📍 Eventos por localização
- 🗂️ Categorias de eventos
- ❤️ Favoritar eventos
- 👤 Autenticação de usuários
